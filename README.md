# Gestionnaire de tâches

Application web développée en Python avec Flask.

## Fonctionnalités
- Création de compte
- Connexion / déconnexion
- Gestion de tâches (ajout, modification, suppression)
- Priorités et dates limites
- Ajout de notes
- Personnalisation de l'interface

## Technologies
- Python
- Flask
- SQLite
- HTML / CSS
